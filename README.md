# Nuryadi-Eka-Data-Analisis-KKA

# Analisis dan Pertanyaan :
1. Mapel mana yang memiliki rata-rata nilai tertinggi?
Matematika memiliki nilai rata-rata yang tinggi dibandingkan mapel lainya.

2. Mapel mana yang memiliki nilai terendah?
Nilai terendah ditemukan pada Bahasa Indonesia dan Fisika, yaitu sekitar 75.

3. Bagaimana visualisasi membantu dalam memahami data?

Visualisasi membantu untuk:

- Dapat Melihat pola data
- Membandingkan nilai antar mapel
- Menyederhanakan data daripada hanya melihat tabel angka

# Refleksi Siswa

1. Apa hal baru yang kamu pelajari dari kegiatan analisis dan visualisasi data?
- Membaca dataset menggunakan Pandas
- Menghitung mean, median, dan modus
- Membuat grafik bar chart dan boxplot
- Menginterpretasikan data menggunakan visualisasi

2. Kesulitan apa yang kamu alami dalam membuat grafik?
- Mengatur delimiter CSV agar dapat dibaca Python
- Memilih grafik yang tepat
- Mendesain grafik agar rapi dan mudah dipahami

3. Menurtu kamu AI apa membantu dalam analysis sebua data?
- Memperbaiki error coding
- Memberikan saran visualisasi
- Menjelaskan hasil analisis
- Mempercepat proses pengolahan data
